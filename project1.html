AMAZON PRIME VIDEO ANALSIS

Project Overview :-


Leveraging my data analysis and visualization skills, I conducted an in-depth analysis
of Amazon Prime Video content. This project involved extracting, cleaning, and analyzing
data to uncover insights about the platform's performance, viewer preferences, and content trends.

Key Responsibilities :-


Data Extraction and Cleaning: Collected data from multiple sources and performed data cleaning to ensure accuracy and consistency.

Data Analysis: Used Python and SQL for data manipulation and analysis to identify key trends and patterns.

Visualization with Power BI: Created interactive dashboards and visualizations in Power BI to present findings in a clear and engaging manner.

Insights and Recommendations: Provided actionable insights and recommendations based on the analysis to help improve content strategy and viewer engagement.

Highlights :-


Content Performance: Analyzed viewer ratings and reviews to identify top-performing genres and titles.

Viewer Preferences: Studied viewing habits and preferences to understand audience demographics and preferences.

Trend Analysis: Examined trends over time to forecast future content performance and viewer interest.

Interactive Dashboards: Developed dynamic Power BI dashboards for real-time data exploration and decision-making support.
